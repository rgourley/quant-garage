# smart-money-cluster

Workflow composite: manager-portfolio-diff across a curated cohort of
9 well-known funds. Aggregates initiations, adds, and exits by
issuer, surfaces cross-fund clustering.

## Quick start

```bash
python3 examples/run-smart-money-cluster.py --min-funds 2
```

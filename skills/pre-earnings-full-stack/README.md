# pre-earnings-full-stack

Workflow composite: earnings-blackout + event-study + guidance-tracker
+ analyst-tracker + mc-portfolio-simulator. One command to answer
"should I trade this print, and how do I size it?"

## Quick start

```bash
python3 examples/run-pre-earnings-full-stack.py --ticker NVDA --proposed-weight 0.10
```

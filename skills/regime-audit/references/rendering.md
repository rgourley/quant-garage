# Rendering: regime-audit

Composite note. Layout:

1. Header (3 lines: identity + verdict + regime counts)
2. Per-ticker table (fixed 8 columns)
3. Take + caveats

Fixed column widths so each row is scannable. Numeric alignment on
Hurst, change-point count, confidence, segment stats.

Verdict tags:
- `BROAD REGIME SHIFT`
- `LOCALIZED REGIME SHIFT`
- `TREND-DOMINATED REGIME`
- `MEAN-REVERSION REGIME`
- `mixed / stable regime`
